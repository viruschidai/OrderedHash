OrderedHash
===========

An ordered hash for node.js

Usage
-----

### Init
var hash = new OrderedHash();

### Add element
hash.set('key1', 'value1');

### Remove element
hash.remove('key1');

### Check length
var length = hash.length;
or
var length = hash.size();


Test
----
